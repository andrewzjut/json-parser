一、个人贷前审核API介绍

个人贷前审核服务是信贷云产品面向信贷机构推出的一项反欺诈服务，专门针对借贷申请环节。此项服务可以帮助平台验证申请材料、发现借贷者的不良信用记录以及多平台负债信息；此外结合信贷云产品专有的线上动态反欺诈模型，可以有效的识别团伙骗贷、机构代办等行为，从而帮助平台降低风险，减少资金损失。

客户可以按照如下步骤进行API集成：

  • 第一步【必选】：在借贷申请环节，调用同盾个人贷前审核服务的submit接口，获取审核报告编号
  • 第二步【必选】：在借贷申请审核环节，调用同盾个人贷前审核服务的query接口，获取报告详细信息
  • 第三步【可选】：集成同盾专业的设备指纹服务，增强欺诈识别能力（ 点击此处 ，跳转至集成文档）

错误代码表

错误代码 错误描述 备注说明
000 认证失败 因系统认证引起的其他未知错误
001 用户认证失败 合作方标识和合作方密钥认证失败
100 非法参数 因参数引起的未知错误
101 参数不能为空 例如：未传入必填参数
102 参数类型不正确 例如：Integer类型传入了String类型值
103 参数超过最大长度 个别参数有长度限制
104 参数格式不正确 例如：字符串时间参数格式不正确
105 时间参数范围不正确 例如：起始时间大于截止时间
106 枚举值不存在 例如：字符串枚举类型传入了一个不存在的枚举值
107 报告导出数量太大 例如：贷前报告导出超出页面提示的值
200 请求已完成  
204 无响应 已接收请求，但不存在要回送的信息（例如：报告未生成）
206 部分信息 处理（返回）信息只是一部分
400 找不到 因访问资源找不到引起的其他未知错误（例如：报告生成失败）
401 未授权 未取得访问资源的授权
402 需要付款 计费流量已用尽，需要付款后再试
403 禁止访问 即使获取授权也不能访问
404 找不到资源 没有找到对应的资源（例如：找不到报告）
405 方法不允许 请求行中指定的请求方法不能被用于请求相应的资源，例如不支持PUT操作
410 授权已过期 取得的授权已经过期
500 内部错误 因内部执行错误引起的其他未知错误
501 未执行 服务器拒绝执行请求资源
